# AgentLedger 全量规划功能核对矩阵

- 文档版本：v1.0
- 更新时间：2026-03-07
- 目标：把历史规划、当前实现、验证证据与结论状态统一到一份可持续维护的功能矩阵。

## 1. 判定规则

本矩阵统一使用以下结论：

- `已实现且已验证`
- `已实现但缺明确验证入口`
- `部分实现`
- `仅规划/长期蓝图`

判定口径：

1. `docs/01` 定义 V1 基础账本能力。
2. `docs/11` 定义老板确认的企业增强路线。
3. `docs/15` 与 `README` 定义当前主干事实清单。
4. `docs/10` 中超出当前事实清单、且无代码证据的内容，统一记为 `仅规划/长期蓝图`。
