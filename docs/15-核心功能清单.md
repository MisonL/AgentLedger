# AgentLedger 核心功能清单

- 文档版本：v1.0
- 更新时间：2026-03-03
- 目标：定义当前可交付核心能力边界，作为需求评审、联调与验收基线。

## 1. 采集与接入

| 功能项 | 说明 | 入口/实现 |
| --- | --- | --- |
| 多源接入模型 | 统一抽象 CLI/IDE/source 的接入配置 | `apps/control-plane` sources API |
| Source 生命周期 | source 创建、查询、删除、多租户隔离 | `/api/v1/sources*` |
| 连通性测试 | 支持 sourceId 与临时配置两种模式 | `/api/v1/sources/:id/test-connection`、`/api/v1/sources/test-connection` |
| 同步任务编排 | 创建、查询、取消同步任务 | `/api/v1/sources/:id/sync-jobs`、`/api/v1/sync-jobs/:id/cancel` |

## 2. 会话与事件

| 功能项 | 说明 | 入口/实现 |
| --- | --- | --- |
| 会话列表检索 | 按关键词与多维过滤搜索会话 | `/api/v1/sessions/search` |
| 会话详情 | 展示 session 基础信息与统计字段 | `/api/v1/sessions/:id` |
| 事件时间线 | 分页返回会话事件列表 | `/api/v1/sessions/:id/events` |
| 来源追溯 | 会话详情可展示 source trace（sourceId/sourceName/provider/path） | contracts + web-console sessions |
| 导出能力 | sessions 支持 JSON/CSV 导出 | `/api/v1/exports/sessions` |

## 3. 使用量与分析

| 功能项 | 说明 | 入口/实现 |
| --- | --- | --- |
| 热力图统计 | 支持 tokens/cost/sessions 指标切换 | `/api/v1/usage/heatmap` + Dashboard |
| usage 聚合 | monthly/models/sessions 多维聚合 | `/api/v1/usage/*` + Analytics |
| 模型视图 | 按模型维度查看用量与成本 | Analytics 页面模型表格 |

## 4. 定价与成本

| 功能项 | 说明 | 入口/实现 |
| --- | --- | --- |
| 定价目录读取 | 获取当前 catalog 与版本信息 | `/api/v1/pricing/catalog`、`/api/v1/pricing/versions` |
| 定价目录写入 | 更新模型计费参数与版本 | `PUT /api/v1/pricing/catalog` |
| 成本分析联动 | usage 聚合与 pricing 数据配合展示 | Analytics / Pricing 页面 |

## 5. 预算与治理

| 功能项 | 说明 | 入口/实现 |
| --- | --- | --- |
| 预算策略 | 支持 tenant/org/user/model/source 等 scope | `/api/v1/budgets` |
| 阈值分级 | warning/critical/block 三段阈值 | governance 规则引擎 |
| 告警列表 | 支持状态筛选与分页 | `/api/v1/alerts` |
| 告警状态流转 | ack/resolve/release 等治理动作 | `/api/v1/alerts/:id/status` + callback routes |

## 6. 回调与集成

| 功能项 | 说明 | 入口/实现 |
| --- | --- | --- |
| 告警分发 | integration 订阅治理事件并路由到外部通道 | `services/integration` |
| 回调入站 | integration 接收上游 callback 并转发 control-plane | `POST /v1/callbacks/alerts` |
| 回调鉴权 | 双端共享 `INTEGRATION_CALLBACK_SECRET` 校验 header | integration + control-plane |
| 链路回归 | 提供脚本化检查与闭环验证 | `check-callback-stream-binding` / `test:callback-chain-targeted` / `test:e2e-governance-callback-chain` |

## 7. 审计与安全

| 功能项 | 说明 | 入口/实现 |
| --- | --- | --- |
| 审计日志 | 记录关键治理与系统动作 | `/api/v1/audits` |
| 多租户隔离 | 资源与审计按 tenant 隔离 | control-plane repository + tests |
| 回调防重放 | callback_id 幂等处理与跨租户隔离 | integration-callback routes |

## 8. Web Console 最小页面能力

| 页面 | 最小可用能力 |
| --- | --- |
| Dashboard | usage 热力图与指标切换 |
| Sessions | 会话列表、详情、事件时间线、token 分解、来源追溯 |
| Analytics | monthly/models/sessions 聚合查询 |
| Sources | source 列表、创建、连通性测试 |
| Pricing | 定价目录读取与更新 |

## 9. 质量工程基线

1. 本地门禁：`bun run lint`、`bun run test`、`bun run build`。
2. 一致性门禁：`bun run check:support-matrix`、`bun run check:callback-stream-binding`。
3. 回调链路回归：`bun run test:callback-chain-targeted`、`bun run test:e2e-governance-callback-chain`。
4. coverage 门禁：`bun run test:coverage`，当前阈值为 ingestion-gateway>=70%、puller>=70%、integration>=75%、control-plane(All files lines)>=80%。
